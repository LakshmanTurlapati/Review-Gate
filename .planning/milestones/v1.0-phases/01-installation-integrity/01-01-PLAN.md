  <action>In `V2/install.sh`, replace the direct commands `pip install mcp>=1.9.2 Pillow>=10.0.0 asyncio typing-extensions>=4.14.0` and `pip install faster-whisper>=1.0.0` with venv-targeted installs that cannot treat `>` as redirection.   <files>V2/install.sh</files>
  <read_first>V2/install.sh</read_first>
  <action>Add a Bash helper near the top of `V2/install.sh` that runs a command with a timeout using this exact fallback order: `gtimeout`, then `timeout`, then a `python3` `subprocess.run(..., timeout=seconds)` fallback when neither binary exists. Use that helper for both current timeout call sites: the SoX microphone probe and the MCP smoke test. Also move the SoX test output path off the hardcoded `/tmp/sox_test_$$.wav` string and derive it from `python3 -c 'import tempfile; print(tempfile.gettempdir())'` so the validation path stays consistent with the rest of the installer. Keep both validations warning-only; they must not become hard install blockers.</action>
  <verify>bash -n V2/install.sh && ! rg -n 'timeout 3s|timeout 5s' V2/install.sh && rg -n 'gtimeout|subprocess.run' V2/install.sh</verify>
  <acceptance_criteria>

    - The microphone probe and MCP smoke test both use the same portable timeout helper.
  </acceptance_criteria>
  <done>Default macOS can execute the POSIX installer end to end without installing GNU coreutils or patching the script manually.</done>
